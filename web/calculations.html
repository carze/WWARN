Instructions

Calculations for sample size and prevelance can be downloaded by using the "Download" button below. Four sets of files are generated and packaged into the tar format and may be extracted using an archiving utility such as WinRAR.

Once the tar file has been downloaded and extracted four tab-delimited files should be produced:

  • sample_size_by_age.txt
  • sample_size_by_study.txt
  • prevelance_by_age.txt
  • prevelance_by_study.txt
